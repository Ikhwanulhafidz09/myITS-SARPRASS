**myITS Sarpras** adalah aplikasi mobile berbasis Flutter yang dirancang untuk mempermudah peminjaman sarana dan prasarana di lingkungan kampus. Aplikasi ini memungkinkan mahasiswa untuk melihat ketersediaan ruangan, melakukan peminjaman, dan mengelola profil pengguna secara real-time.
